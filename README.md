# 🚀 Prompt Llama 3 Like a Pro

This project explores and evaluates Meta’s **Llama 3 (8B-chat-hf)** large language model using a structured set of prompts across multiple domains such as reasoning, coding, creativity, and general knowledge.

---

## 📌 Project Objective

The main goal of this project is to analyze how well Llama 3 performs in:

- General knowledge question answering
- Creative writing (poems, storytelling)
- Code generation (Python & C++)
- Software design tasks
- Multi-parameter structured prompts
- Multi-step reasoning (chain-of-thought)

Additionally, the model is compared with other LLMs (e.g., Gemma) to evaluate relative performance.

---

## 🧠 Model Used

- Meta LLaMA 3 (8B Chat HF)
- Loaded via Hugging Face Transformers
- Inference optimized with GPU support (PyTorch)
